# AI Excuse Generator (NLP)

An AI-powered chatbot that generates context-based excuses using transformer-based NLP models.

## Features
- Generates excuses based on user input
- Multi-language support using translation models
- Text-to-speech (TTS) integration
- Scenario-based customization
- Smart excuse ranking

## Tech Stack
- Python
- Hugging Face Transformers (T5, MarianMT, BART)
- Coqui TTS

## Project Structure
- `project.ipynb` – Main implementation notebook

## How It Works
- User inputs a scenario
- NLP model generates an appropriate excuse
- Optional translation and speech output

## How to Run
1. Open the notebook in Jupyter/Colab
2. Install required libraries
3. Run all cells
